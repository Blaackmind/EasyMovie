# EasyMovie
Aplicativo mobile integrado a API publica "The Movie Database"
O EasyMovie é uma plataforma prática e intuitiva para amantes de filmes e séries explorarem novidades, pesquisarem títulos e salvarem seus favoritos. Desenvolvido como parte do projeto trimestral de desenvolvimento mobile, o app aplica boas práticas de programação, gerenciamento de estado com hooks do React, autenticação de usuários, consumo de APIs públicas e estilização moderna com React Native Paper.
Desenvolvedores:Amanda Kelory, Ana Panullo e Felipe Martins
